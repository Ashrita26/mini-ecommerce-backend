# Mini E-Commerce Backend

## Problem Statement

This project is a menu-driven Mini E-Commerce Backend System developed using Python and SQLite. It manages customers, products, carts, orders, and payments for an online shopping system.

## Features

* User Management
* Product Management
* Shopping Cart
* Order Placement
* Payment Management
* Search Operations
* Inventory Tracking

## Technologies Used

* Python 3
* SQLite
* SQL
* VS Code
* Git & GitHub
